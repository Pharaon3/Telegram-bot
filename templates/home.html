TELEGRAM - MENU - BOT


Create Telegram bot using Botfather and set BOT_API_KEY for BOT script

1. Start a new conversation with the BotFather.

2. Send /newbot to create a new Telegram bot.

3. When asked, enter a name for the bot.

4. Give the Telegram bot a unique username. ...

5. Copy and save the Telegram bot's access token for later steps.

6. Open the 'bot_config.py' file and set the BOT_API_KEY as access token

Enjoy!
